# Automated Web Form Tester (single-layer: scheduler + runner)
A Playwright-based synthetic monitoring script that performs automated end-to-end validation of a live web form by simulating real user interactions, verifying dynamic UI state transitions, and alerting on unexpected application behavior.

# Requirements
Built on Python 3.13.7
- Install:
  - `pip install playwright requests`
  - `playwright install chromium`
